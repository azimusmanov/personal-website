# azimusmanov.com

Personal website of Azim Usmanov — Computer Engineering student at Northwestern University.

Built with Next.js and Tailwind CSS. Deployed on Cloudflare Pages.

## Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Fonts**: Geist Sans + Geist Mono
- **Deployment**: Cloudflare Pages
